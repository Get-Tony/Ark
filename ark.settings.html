ark.settings.get_projects_dir() → str

Check for a projects directory environment variable.

Custom projects directory can be set with the environment variable ARK_PROJECTS_DIR. If not set, the default projects directory is $HOME/ark_projects.

If a defined custom projects directory does not exist, Ark will exit with an error.
