ark.core.run

Ark - Playbook Execution.

ark.core.run.find_playbooks(project_name: str) → List[str]

Find all playbooks for a project.

Rules:
  • Playbooks are located in the project directory under the ‘project’

    subdirectory.

  • Playbooks must have a ‘.yml’ or ‘.yaml’ extension.

Parameters:

project_name (str) – The name of the project.

Returns:

A list of playbooks.

Return type:

List[str]

ark.core.run.get_playbook_path(project_name: str, playbook_file: str) → Path | None

Get the path to a playbook.

Parameters:

  • project_name (str) – Project name.

  • playbook_file (str) – Playbook filename.

Returns:

Path to the playbook or None if it does not exist.

Return type:

Optional[Path]

dict[str, str]

ark.core.run.run_ansible_playbook(project_name: str, playbook_path: Path, rotate_artifacts: int, limit: str, extra_vars_dict: dict[str, str], verbosity: int = 0) → Any

Run an Ansible playbook.

Parameters:

  • playbook_path (Path) – Path to the playbook.

  • rotate_artifacts (int) – Artifact rotation limit.

  • limit (str) – Ansible limit.

  • extra_vars_dict (dict[str, str]) – Extra vars.

  • verbosity (int, optional) – Ansible verbosity. Defaults to 0.

Returns:

Ansible runner result.

Any
